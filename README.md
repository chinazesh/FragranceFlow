# FragranceFlow

FragranceFlow is a Python-based management system designed for perfume enthusiasts and resellers. It simplifies inventory tracking, volume management, and price calculation for decanting (selling fragrance samples).

## Key Features
Inventory Management: Add and track your fragrance collection using Object-Oriented Programming (OOP).
Decant Calculator: Automatically calculate prices for  5ml and 10ml samples based on the initial bottle cost.
Stock Alerts: Get notified when perfume levels are low.
Data Persistence: All your collection data is saved and loaded from local files (JSON/TXT).
Action Logging: Every transaction is logged using Python Decorators for transparency.

## Requirements (AITU Course Criteria)
This project implements the following technical concepts:
Data Structures: Utilization of Lists, Dictionaries, and Sets for efficient data handling.
OOP: Classes for Perfume, Inventory, and Transaction.
File I/O: Reading and writing collection data to ensure it is not lost after closing the app.
Decorators: Custom decorators for logging and input validation.
Modules: Use of json, os, and datetime libraries.

## Project Structure
main.py: The entry point of the application.
inventory.py: Logic for managing the perfume collection.
data/: Folder containing saved collection files.
proposal.pdf: Detailed project plan and timeline.
